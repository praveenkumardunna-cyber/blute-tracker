Connect to Google Sheets
Your data will be stored permanently in a Google Sheet you own — free, accessible from any device or browser.
1
Create a Google Sheet
Go to sheets.google.com → create a new spreadsheet → name it BLUTE Task Tracker
2
Open Apps Script
In your sheet → click Extensions → Apps Script → delete all existing code → paste the apps_script.gs file contents provided → click Save (Ctrl+S)
3
Deploy as Web App
Click Deploy → New deployment
Type: Web app
Execute as: Me
Who has access: Anyone
Click Deploy → authorize when prompted → copy the Web App URL
4
Paste your Web App URL below
It looks like: https://script.google.com/macros/s/AKfycb.../exec
