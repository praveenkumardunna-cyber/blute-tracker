Legend — What each letter means
R — Responsible
A — Accountable
C — Consulted
I — Informed
A/R — Accountable + Responsible
Responsible (R): Does the actual work. Multiple R's allowed per task.
Accountable (A): Ultimately answerable — only ONE per task.
Consulted (C): Expert input needed before decisions (two-way communication).
Informed (I): Kept in the loop after decisions (one-way communication).
A/R: Same person is both Accountable and Responsible — typical for founder-led tasks.
